# Video Conferencing App

A full-stack real-time video conferencing application built using the MERN stack and WebRTC. The app enables users to create and join secure meeting rooms with seamless peer-to-peer audio/video communication and a responsive user interface.

---

## 🚀 Features

- Real-time video and audio calling
- Secure room creation and joining
- Peer-to-peer communication using WebRTC
- Responsive and modern UI
- Multiple participant support
- Mute / unmute functionality
- Camera on / off controls
- Dynamic room-based communication
- Socket-based real-time signaling
- Fast and scalable MERN stack architecture
